# 双轨打分与背离检测

这是本 skill 的方法论核心。把"热"拆成两条独立轨道分别量化，再比较两轨的相对高低判断热度质量。**两轨分开打 1-5 分 + 背离判定**这两件事不能省。**全程不使用历史分位概念**——评分用 1-5 分制，标准见下。

---

## 一、为什么要双轨

"热"是一个被混用的词。一个板块可能：
- 消息大量涌现但买盘不足（**信息热、行情冷**）——题材还没启动，或者利好出尽利好兑现回落；
- 闷声大涨但没什么消息（**行情热、信息冷**）——纯买盘 / 量能驱动，缺基本面催化；
- 消息和资金同步共振（**双热**）——趋势最扎实；
- 两边都冷清（**双冷**）——没有明确机会。

把两轨拆开，才能回答用户真正关心的问题："这波热度的质量好不好、能不能持续。"单一一个"热度分"会把上面四种完全不同的状态混成一个数字，失去判断力。

---

## 二、评分总则（1-5 分制）

综合 / 信息 / 行情热度都用 **1-5 分**，每分对应一个档位（与脚本、飞书文档展示完全一致）：

| 分数 | 档位 | 含义 |
|---|---|---|
| 1 | 冷清 | 几乎没有热度 |
| 2 | 温和 | 有一点但很弱 |
| 3 | 活跃 | 中等、强弱参半 |
| 4 | 高热 | 明显偏热 |
| 5 | 过热 | 极热 / 见顶特征 |

打分时**对照下面每一档的具体口径**来定档，做到"评分科学、每个分数都可能达到"。打分是综合判断而非机械加总：先看各项落在哪一档，再给一个最能概括整体的整数分。

板块价格、成交等输入必须是目标板块自身数据；10 股样本只用于代表股表现、分化和股票 PE(TTM)，不替代板块级指标。

---

## 三、行情热度 1-5（看价格 / 成交量能 / 代表股表现 / 估值位置）

观察四件事，与模块1「现在有多热」的四个维度、飞书文档行情轨标签（"价格 / 成交 / 代表股 / 估值的强度"）完全对齐：（1）**价格涨跌**（超额动量、当日涨跌与近7日反弹力度）；（2）**成交量能**（成交额、换手、量比的相对变化，是放量还是缩量）；（3）**代表股表现**（10 只代表股普涨还是分化、龙头领涨还是杀跌）；（4）**估值位置**（代表股 PE(TTM) 在自身区间偏高还是偏低）。

| 分 | 档位 | 典型特征（四个维度的综合表现） |
|---|---|---|
| **5** | 过热 | 放量大涨 + 代表股普涨甚至涨停梯队成型 + 超额收益显著，价格、量能、代表股**几乎全部强确认**；常伴随估值抬到偏高位置、情绪亢奋的见顶风险。 |
| **4** | 高热 | 明显放量 + 超额收益为正 + 多数代表股上行，**多数维度确认、趋势偏强**，但尚未到全面亢奋。 |
| **3** | 活跃 | 成交中性偏强、价格有波动但方向不明、代表股涨跌分化，**强弱参半**（部分维度确认、部分背离），或处于刚反弹 / 刚降温的过渡期。 |
| **2** | 温和 | 成交平淡、价格窄幅波动、代表股多数滞涨，**仅个别维度偏暖**，整体交投不活跃。 |
| **1** | 冷清 | 缩量 + 价格走弱或滞涨 + 代表股普跌、上涨面窄，**几乎无积极信号**。 |

> **估值位置**是"热度可持续性"的反向维度：估值越贵，越不支撑行情热度继续上行，越容易使"估值位置"这一维度判为**背离**，并进入风险模块——它压热度，而不是抬热度。

---

## 四、信息热度 1-5（看催化的密度与强度，不分方向）

观察催化事件的**密度**（多不多）与**强度 / 级别**（官方政策 > 行业事件 / 大额订单 > 一般新闻 > 传闻）。

| 分 | 档位 | 典型特征 |
|---|---|---|
| **5** | 过热 | 重大催化密集叠加（政策 + 产业 + 事件多重），全市场高度关注、研报与媒体高度密集。 |
| **4** | 高热 | 出现**明确的强催化**（重磅政策 / 大额订单 / 标志性事件 / 密集利空），关注度明显升温。 |
| **3** | 活跃 | 有**若干可核验催化**，话题有一定热度但未达高度密集。 |
| **2** | 温和 | 仅零散消息或预期，**缺乏强催化**。 |
| **1** | 冷清 | 基本无新增催化，关注度低。 |

> 两个要点：
> 1. **信息热度只看"多不多、强不强"，不分利好利空方向**——方向在事件归因（模块2）里体现。一个被大量**利空**消息密集的板块，信息热度也可以是 4-5 分。
> 2. **质量影响打分**：催化的可信度等级直接影响信息热度——一级（一手 / 权威原文）催化可足额计入；二级来源（媒体原创、行业第三方）可计入但偏保守；**三级平台 / 自媒体 / 无可打开出处的传闻只能作为回溯线索，不进入模块2 展示，也不计入信息分**（见 `references/data_rules.md`）。很多"虚火"板块就是信息分被未经核实的传闻撑高、却经不起证据检验——按可信度分级打分能避免这种虚高。

---

## 五、综合热度 1-5

两轨加权合成，行情轨权重略高（价格和成交更难造假）：

```
综合热度 = round( 行情热度 × 0.55 + 信息热度 × 0.45 )   # 结果夹到 1-5
```

例如行情 2 + 信息 4 → round(2×0.55 + 4×0.45) = round(2.9) = **3（活跃）**。这种加权能保证 1、2、3、4、5 每个分数都可能达到。**开篇核心结论必须给出这个 1-5 的综合分 + 档位词**，不要用"较热 / 待验证"含糊带过。

---

## 六、双轨背离检测（最关键的一步）

比较**信息热度**与**行情热度**，落到四象限。**阈值：某一轨 ≥ 3 分（活跃及以上）即视为"热"，< 3 视为"冷"。**

| | 行情热（≥3） | 行情冷（<3） |
|---|---|---|
| **信息热（≥3）** | **双热 → 强趋势共振**<br>消息与资金互相印证，是质量最高的热度，持续性相对最强。但要警惕是否已进入过热区间（综合 5 分 + 估值偏高）。 | **信息热 / 行情冷 → 题材未启动 或 利好兑现回落**<br>有故事没买盘。两种情形：（1）题材刚出还没发酵（潜在机会）；（2）利好兑现后资金离场（风险）。**利空密集导致的超跌修复初期也常落在此格**——要结合事件时点判断是哪种。 |
| **信息冷（<3）** | **行情热 / 信息冷 → 纯买盘驱动**<br>价格和量能先行，但缺乏公开催化与景气验证。持续性存疑，最怕证伪——一旦驱动逻辑反转，缺基本面支撑的品种会率先回撤。 | **双冷 → 低关注**<br>关注度和买盘都不足，板块处于沉寂期，缺乏明确的短期机会。 |

每种背离都要输出三样东西：
1. **背离类型**（如"信息热 · 行情冷"）；
2. **一句话判定**（如"利空已被定价、行情处于超跌修复初期，持续性待验证"）；
3. **对持续性的含义**（展开说明这意味着什么、最该警惕什么），这是用户最想要的结论。

> **背离的核心存疑 → 模块4 的信号来源**：第 3 点里"决定持续性的那个关键变量"（如"行情热度能否补上来"），正是模块4「接下来盯什么」最该转成可证伪信号的对象（如"板块能否连续放量 + 资金净流入"——资金类信号属模块4 盯盘对象，不进模块1 的四维度打分）。写完背离判定后，把这个核心存疑接到模块4，能保证盯盘信号指向真正决定方向的未来变量，而不是泛泛而盯；模块4具体写法见 `references/chat_contract.md`。

> payload 里 `divergence.type` 可直接给定；脚本也能根据 `info_score` / `market_score` 与阈值 3 自动判定双轨背离类型。两者取其一即可，但要保证与正文结论一致。
