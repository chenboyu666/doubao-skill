## 七、稳定性约束（所有脚本必须遵守）

1. **IIFE 包裹**：`(function(){ ... })();`，不污染全局。
2. **try-catch**：主逻辑包在 `try { ... } catch(e) { console.error(e); }` 内。
3. **DOM 先检查**：`if (!el) return;`
4. **禁止 DOMContentLoaded**：Shadow DOM 环境下不触发，直接执行。
5. **自包含**：禁止访问外部变量，所有数据在块内定义。
6. **动态公式用 Unicode**：禁止 LaTeX，详见§八。
7. **CDN 选择**：使用 ECharts 等外部库时优先使用 jsDelivr，**严禁使用 BootCDN**。引入外部库必须做 `typeof lib !== 'undefined'` 检查，加载失败时**渲染可读的回退文本**（如"图表库加载失败，已回退为文本说明"），不允许留空白。
8. **严禁地图可视化**：**绝对禁止**调用任何地图相关的组件或库（如 Leaflet、ECharts 的 `geo/map` 组件等），**绝对禁止**绘制行政区划、瓦片地图、坐标点或地理轨迹。如遇地图需求，一律回退为纯文本解释。
9. **解析期错误预防（致命错误防范）**：所有 renderer 代码在输出前必须做语法级自检，禁止出现会在解析期直接中断的写法，包括非法 `const/let` 赋值、给构造器表达式赋值、未闭合括号/字符串、残缺的对象字面量、截断的字符串拼接。try-catch 不能捕获解析期错误，因此宁可使用更简单的 SVG/Canvas，也不要输出无法解析的复杂脚本。
10. **防死循环与高度约束（致命错误防范）**：
    - **现象与原理**：代码运行在高度自适应的 iframe（带 ResizeObserver）中，**绝对禁止在任何地方使用 `100vh`、`min-height: 100vh`、`height: 100%` 或在根容器写死大高度（如 `height: 720px`）**。`vh` 会与 iframe 自动调整机制冲突导致无限拉长；写死固定大高度会在内部响应式折叠（换行）时导致严重的内容溢出和重叠！
    - **强制规范**：
      - 如果需要设置最小高度，**只能使用 `min-height` 配合固定像素值**（例如：`min-height: 400px;`），绝对不能用相对视口的单位，也绝不能用 `height` 写死根容器高度。
      - 所有的布局容器必须加上 `box-sizing: border-box;`，防止 padding 撑破容器引发高度重算。
      - 容器高度必须默认由内部内容自然撑开（`height: auto`），以完美适配移动端的折叠换行。

**降级策略**：

| 异常 | 处理 |
|------|------|
| CDN 加载失败 | `typeof lib !== 'undefined'` 检查，失败时**渲染可读的回退文本**，不留空白 |
| JS 运行时报错 | `catch` 中输出纯文本提示，不留空白 |
| 解析风险过高 | 改用纯 SVG / Canvas / HTML+CSS |
| 对比度不达标 | 回退纯文本或深色文字 + 浅色底 |
| 地图类需求 | 禁止地图 renderer，回退纯文本说明 |
| 无可靠数据 | 输出空态/示例结构 + 明确标注"未取得可靠数据" |

## 八、数学公式约束

**静态公式**：用 MathJax，CDN 置于 ` ```html type="renderer" ` 内部。

**动态公式**（JS 实时更新）：严禁 LaTeX（iframe 隔离+innerHTML 不触发重排），改用 Unicode：

| 含义 | ❌ 禁止 | ✅ 改用 |
|------|---------|---------|
| 分式 | `\(\frac{a}{b}\)` | `a/b` 或 `<sup>a</sup>⁄<sub>b</sub>` |
| 希腊字母 | `\(\pi\)` `\(\omega\)` | `π` `ω` |
| 运算 | `\(\sum\)` `\(\sqrt{x}\)` | `Σ` `√x` |
| 上下标 | `\(x^2\)` `\(x_n\)` | `x²` `x<sub>n</sub>` |
| 关系符 | `\(\approx\)` | `≈ ≤ ≥` |

```javascript
// 动态公式示例：傅里叶
function updateFormula(n) {
  var terms = [];
  for (var k = 1; k <= n * 2 - 1; k += 2) {
    terms.push('sin(' + k + 'ωt)/' + k);
  }
  formulaDiv.innerHTML = 'f(t) = (4/π) × [' + terms.join(' + ') + ']';
}
```
