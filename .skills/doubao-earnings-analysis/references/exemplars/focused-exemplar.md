# XX 公司毛利率下降不是经营恶化，而是 B 产品放量带来的结构性稀释

## 营收和毛利润都在涨但毛利率下降——成本涨得比收入快，指向 B 产品占比提升带来的结构性稀释，叠加上游议价劣势和竞争定价约束

这个现象初看矛盾，但数学上完全自洽：成本增速（+92%）比收入增速（+68%）快了 24 个百分点，每多赚 1 元收入对应的成本变多了——不是「生意变差了」，而是「赚钱的方式在变」。核心驱动是 B 产品占比快速提升，而 B 的成本结构天然更重、公司在上游没有议价权、在下游没有定价权，三重约束叠加使得毛利率随 B 产品放量而被结构性拉低。

## 1. B 产品占比快速提升——为什么规模做大了毛利率反而不能改善？

20X6Q1 营收 40.3 亿元、同比 +68%，营业成本 17.9 亿元、同比 +92%。毛利润 22.4 亿元同比 +53%，但因为成本增速快于收入增速，毛利率从 61.2% 降至 55.6%，同比 -5.6pct。

| 指标 | 20X5Q1 | 20X6Q1 | 同比变化 |
|------|--------|--------|---------|
| 营收 | 24.0 亿 | 40.3 亿 | +68% |
| 营业成本 | 9.3 亿 | 17.9 亿 | +92% |
| 毛利润 | 14.7 亿 | 22.4 亿 | +53% |
| 毛利率 | 61.2% | 55.6% | -5.6pct |
| B 产品占比（测算） | ~25% | ~45% | +20pct |

成本增速（+92%）显著快于收入增速（+68%），矛盾的核心在第五行：B 产品占比从约 25% 跃升至约 45%——这正是「毛利润在涨、毛利率在跌」的结构性根源。

成本为什么增速更快？卖方测算显示 B 产品收入占比从 20X3 年不足 20% 升至 20X5Q3 的 40%-45% 区间。A 产品是 IP 授权模式，硬件物料占售价比重低；B 产品需要搭配高带宽内存和先进封装，物料成本占比明显更高。

**但为什么 B 产品的毛利率不能随着出货量增长而改善？**

- **上游 HBM 供应商高度集中**（全球仅海力士/三星/美光三家），XX 公司作为中等规模采购方，在 HBM 产能紧缺期（当前 HBM 供需缺口约 5%）完全没有议价能力——量再大也是「排队拿货」，不是「压价采购」
- **先进封装产能被头部客户锁定**，XX 公司在代工厂产能分配优先级中排位靠后，拿到的产能有限且价格弹性小
- **B 产品仍在生态建设和市场渗透早期**，公司需要以有竞争力的价格建立客户基础——这个阶段提价 = 丢客户 = 生态无法积累

三重约束叠加意味着：B 产品的低毛利率不是「暂时亏钱做量」，而是当前竞争格局和供应链位置的必然结果。短期内不会因为出货量增长而显著改善。

## 2. 上游制造成本在涨——为什么公司不能转嫁或规避？

公司在 20X5 年报中明确提到「上游原材料涨价、代工厂封装测试产能紧张」对毛利率构成压力。B 产品采用的先进工艺和封装产能紧张程度高于 A 产品使用的成熟工艺，成本传导更直接。

**为什么不能切换供应商或自建？**

- 先进制程代工全球可选供应商极少，XX 公司受地缘限制后选择更窄——事实上能用的只有有限几家，没有「货比三家」的空间
- 先进封装（CoWoS 类工艺）技术壁垒高，自建周期需要 2-3 年、数十亿资本投入，远水解不了近渴
- B 产品的芯片设计与特定代工工艺深度绑定，切换供应商意味着重新流片验证（6-12 个月 + 数亿费用），在当前放量窗口期不可接受

结论：上游成本压力是结构性的——不是「今年涨了明年回落」的周期波动，而是先进工艺/封装产能全球性紧缺的长期状态。公司作为中等规模客户，在供应链中的位置决定了它是成本的「承受者」而非「议价者」。

## 3. 竞争定价约束了售价弹性——为什么公司不能提价对冲成本？

竞争对手甲已建立完整软件生态体系，开发者迁移成本极高。竞争对手乙在实现盈利后开始激进降价争夺份额。XX 公司定价空间受两面夹击。

**为什么不能通过差异化维持定价？**

- **竞对甲的生态锁定**：甲的开发工具已成行业标准，XX 公司虽然兼容甲的编程生态（迁移成本低），但硬件性能仍存在代差。客户选择 XX 公司是因为「可用 + 可得」，不是因为「更好」——这种替代逻辑天然限制定价：必须比甲便宜才有吸引力
- **竞对乙的价格竞争**：乙已完成扭亏、开始用价格换规模，XX 公司在同一批国内客户中竞争，不跟进则丢单
- **国产替代红利的天花板**：政策要求确实提供了「准入门票」，但客户仍然要求国产方案比进口替代方案有价格优势（通常 20-30% 折让）——这是价格上限，不是定价自由度

放量阶段以价换生态是合理的商业选择——但代价就是毛利率承压。公司当前的定价策略是「跟随型」而非「引领型」，这由竞争格局决定，不是管理层的失误。

**时间维度看，同比在掉但环比已企稳。** 毛利率季度轨迹：20X4 全年 63.7% → 20X5Q3 60.0% → 20X5Q4 约 53.4% → 20X6Q1 55.6%。下行趋势从 20X5 年就开始了，但 20X6Q1 环比回升 +2.2pct——Q4 是阶段低点。

**对公司经营健康度的判断：** 毛利率从 63% 降到 55% 不是「赚钱能力变差」，而是商业模式从「高毛利率-小规模 A 产品」向「中毛利率-大规模 A+B 双产品」的主动切换。关键验证指标是毛利润绝对额——当前同比 +53%，说明规模扩张的速度仍然跑赢了毛利率下滑的幅度。但这个平衡有边界：如果 B 产品占比继续提升而 B 自身毛利率不能改善，混合毛利率可能进一步下行至 50% 以下——届时即使收入高增，毛利润增速也会明显放缓。
