# B 金融型 Playbook

## 适用边界

收入来自金融资产、负债、保费、佣金、交易或投资收益。金融型没有毛利率概念，禁止计算或引用 `gross_margin_pct`。

## 银行

收入主拆分：

`营收 = 净利息收入 + 手续费及佣金净收入 + 其他非息收入`

`净利息收入 ≈ 平均生息资产 × 净息差`

盈利分析顺序：

1. 拨备前利润是否增长。
2. 信用成本和减值计提是否变化。
3. 不良率、关注类、逾期、拨备覆盖率是否支撑利润质量。
4. ROE/ROA 变化来自息差、杠杆、成本收入比还是拨备。

前瞻指标：净息差走势、贷款投放、存款成本、不良率、关注类、逾期、拨备覆盖率、资本充足率。

## 保险

寿险拆分：

`NBV ≈ 新单保费 × 价值率`

产险拆分：

`承保利润 ≈ 保费 × (1 - 综合成本率)`

利润必须拆开承保利润和投资收益。不要把权益市场上涨带来的投资收益写成承保改善。

前瞻指标：新单保费、NBV、价值率、代理人产能、银保渠道结构、综合成本率、赔付率、费用率、投资收益率。

## 券商

收入按分部加总：

`营收 = 经纪 + 投行 + 资管 + 利息 + 自营`

自营随市场 beta 波动，禁止硬做量价拆分。利润分析要同时看分部贡献、杠杆、ROE、风险资本占用。

前瞻指标：市场成交额、两融余额、IPO/再融资项目储备、资管规模、权益市场表现、债券市场波动。

## 租赁与其他金融

收入看资产规模、收益率和资金成本。利润大增先问是不是拨备、减值或资产处置变化，而不是默认经营改善。

## 常见对质对（阶段 2 逐对筛选，处置规则见 analysis-moves.md 动作三）

| 对质对 | 展开门槛 |
|---|---|
| 营收增速 vs 拨备前利润增速 vs 归母净利增速 | 三者间任意两个方向相反或幅度差 >10pct（利润是否靠少提拨备/释放准备金） |
| 不良率方向 vs 关注类+逾期率方向 | 不良下降但关注类/逾期上升（资产质量前瞻指标与表观指标背离） |
| 净息差 vs 净利息收入增速 | 息差收窄但利息收入高增（靠规模扩张掩盖定价恶化，看可持续性） |
| 保险：净利润方向 vs 承保利润方向 | 方向相反（利润靠投资收益支撑，需剥离权益市场 beta） |
| 保险：NBV 增速 vs 新单保费增速 | 缺口 >10pct（价值率与规模哪个在驱动，结构变化方向） |
| 券商：自营收入占比变动 | 单季占比变动 >10pct（利润的市场 beta 成分，禁止写成经营能力改善） |
| 中收增速 vs 监管/费率政策方向 | 两者矛盾（指向一次性因素或口径调整） |

## 弱拆解代理链（披露不足时）

银行季报不披露分项息差/成本明细、保险季报不披露 NBV 时，代理链：监管行业数据（央行/金融监管总局口径）+ 可比同业同期披露 + 券商模型假设（具名转引）。只有单一来源时结论降为「单一解释」档（档位规则见 `../analysis-moves.md`），禁止用「资产质量改善」「负债成本优化」等确定性措辞替代验证。

## 红线

- 禁止毛利率。
- 利润大增先检查是否少提拨备、准备金或减值。
- 媒体新闻不能当资产质量或业绩事实。
- 不输出自有评级与目标价；机构观点仅具名转引。
