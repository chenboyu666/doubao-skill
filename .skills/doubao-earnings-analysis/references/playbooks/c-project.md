# C 项目交付型 Playbook

## 适用边界

收入来自项目、合同、工程、重大设备或地产结算。收入确认受履约进度和结算节奏影响，当期量价拆分通常无意义。

## 收入主拆分

基本思路：

`收入 ≈ 期初在手订单 × 本期确认进度 + 本期新增订单当期确认部分`

分析时优先回答：

- 新签订单同比如何。
- 在手订单对未来收入的保障倍数如何。
- 本期收入变化来自确认节奏、项目结构还是新签变化。
- 重大项目是否影响毛利率和现金流。

## 盈利口径

毛利率受项目结构影响，必须落到具体项目类型或地区。应收账款、合同资产、减值是利润大变量，单独检查。

必须检查：

- 经营现金流和利润是否背离。
- 应收账款、合同资产、存货、合同负债变化。
- 资产减值、信用减值、坏账计提。
- 项目毛利率是否因低毛利大项目集中确认而变化。

## 前瞻硬指标

- 新签订单及同比。
- 在手订单。
- 在手订单 / 年收入保障倍数。
- 重大中标、合同公告。
- 合同负债和预收款。
- 回款周期和应收账款周转。

## 二级标签

### 工程

收入确认与施工进度相关，订单和回款比单季收入更能代表持续性。风险提示必须写回款、垫资、减值机制。

### 军工/船舶

大合同和交付节点决定收入节奏。关注预收款、合同负债、存货、在建项目、产能瓶颈。

### 大型设备

订单到交付有周期。区分新签、在手、发货、验收和收入确认。不要把订单增长直接等同当期收入增长。

### 地产

销售额与结算收入错期 1-2 年，必须分开写。合同负债/预收款是结算蓄水池；必看销售额、去化、土储、净负债率。投资判断主要看销售端，而不是只看本期结算利润。

## 常见对质对（阶段 2 逐对筛选，处置规则见 analysis-moves.md 动作三）

| 对质对 | 展开门槛 |
|---|---|
| 收入增速 vs 新签订单增速 | 方向相反（吃存量订单，未来收入承压；或订单高增但确认滞后） |
| 归母净利 vs 经营现金流 | 方向相反或同向幅度差 >30pct（垫资、回款恶化，项目型最高发） |
| 应收账款+合同资产增速 vs 收入增速 | 高出 >20pct（收入质量，减值前瞻） |
| 毛利率单季跳变 vs 项目结构 | 跳变 >3pct（低毛利大项目集中确认还是结构性改善，必须落到项目类型） |
| 合同负债/预收款方向 vs 收入增速方向 | 方向相反（蓄水池与确认节奏背离） |
| 地产：结算收入/利润方向 vs 当期销售额方向 | 方向相反（错期 1-2 年，禁止用本期结算利润判断经营趋势） |
| 减值计提节奏 vs 利润增速 | 利润增长伴随减值计提骤降（利润含释放成分） |

## 弱拆解代理链（披露不足时）

季报不披露新签/在手订单明细时，代理链：中标公告与重大合同公告累计 + 合同负债方向 + 券商订单跟踪测算（具名转引）+ 行业招投标数据。只有合同负债方向一条时，对「订单景气」的结论降为「单一解释」档（档位规则见 `../analysis-moves.md`）。
