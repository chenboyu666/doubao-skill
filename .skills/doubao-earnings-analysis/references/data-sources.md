# 数据源与搜索路径

## 搜索工具

你有两种搜索方式，按需选用：

| 工具 | 用途 | 适用场景 |
|------|------|---------|
| `seed_finance_search` | 从专业金融数据库搜索 | 财务数据、财报原文、券商研报、一致预期、公司公告 |
| `general_search` | 通用网页搜索 | 行业动态、新闻事件、第三方数据、业绩说明会纪要、竞品信息 |

优先用 `seed_finance_search` 获取财务数字和机构观点；用 `general_search` 补充经营性事实、行业背景和独立验证数据。两者可组合使用。

---

## 原理

慢变量可以用训练先验启动，但必须标注「先验，待验证」。快变量必须当场搜索并落盘：数字、份额、近期事件、预期锚、指引、市场关注点、财报后机构观点都属于快变量。

**先做市场路由**：本文件的检索模板以 A 股为主。港股、美股中概的披露平台、定期报告体系、预期锚来源按 `market-profiles.md` 的市场档案替换（如港交所披露易、SEC EDGAR/公司 IR、moomoo/Yahoo 转引的标普一致预期）。搜集到的来源尽量带原文 URL 落盘——它们就是成品报告的参考文献链接。

## 定期报告与公告

优先级最高，作为财务事实主来源：

- 巨潮资讯网：A 股法定披露平台。
- 上交所/深交所官网；港交所披露易（A+H 及港股）；公司投资者关系页面。

检索模板：

- `{公司名} {报告期} 第一季度报告 巨潮`
- `{股票代码} {报告期} 季度报告 cninfo`
- `{公司名} 业绩快报 {报告期}`
- `{公司名} 投资者关系活动记录表 {报告期}`

## 业绩说明会与投资者关系

优先查：上证路演中心、全景网、公司 IR 活动记录、财经媒体报道的说明会纪要。

引用格式：`公司业绩说明会(YYYY-MM-DD)`；经媒体报道写 `公司业绩说明会(YYYY-MM-DD)，经 XX 媒体报道`。说明会是「管理层信号表」的主要输入，重点记录：量化目标、时间表承诺、口径解释、供需表态。

## 一致预期降级路径

只允许财报发布前的预期作为判定是否达到预期的锚点。取到哪一级用哪一级，并写明：

1. Wind、Choice、iFinD 等终端一致预期。
2. 券商研报转引的一致预期，例如「交银国际引用 Visible Alpha 一致预期(YYYY-MM-DD)」。
3. 财经媒体的财报前瞻，需具名和日期。
4. 都没有：`consensus.available=false`，改用公司指引或历史趋势做锚，并在报告写「未获取一致预期」。

最多检索 5 轮。找不到时降级，不继续消耗注意力。

## 财报后机构观点（条件章节素材）

- 用途：仅供「机构观点汇总」章节具名转引；≥2 家成章，1 家并入段落，0 家写入未获取清单。
- 每条必须记录：机构名、观点要点（评级/目标价/预测调整方向）、报告日期、转引出处。
- 区分角色：财报前报告可作预期锚；财报后点评只能用于机构观点转引，禁止倒用为财报前锚。
- 禁止据机构目标价推导「上行空间」等自有结论；禁止包装成「我们的预测」。

## 独立验证数据（对质与关注清单素材）

公司口径的战略性表述（份额、UE 改善、商业化进展）尽量找独立数据对照；找不到时在关注清单写「建议读者结合 XX 交叉验证」。单独留 2-3 轮搜索预算：

- 竞争对手同期财报/经营数据（同一指标的口径对照）。
- 行业第三方数据（见下表）。

| 领域 | 常用来源 |
|---|---|
| 动力电池 | SNE Research、中国汽车动力电池产业创新联盟、公司公告 |
| 汽车 | 中汽协、乘联会、公司产销快报 |
| 银行/保险 | 定期报告、央行、金融监管总局、月度保费公告 |
| 券商 | 月度经营数据、交易所成交额、两融数据 |
| 地产 | 克而瑞、公司月度经营公告 |
| 建筑/军工 | 中标公告、重大合同公告 |
| 消费 | 行业协会、渠道调研、第三方零售监测 |
| 互联网/云 | 第三方云市场份额报告、QuestMobile、竞品财报 |
| 软件/SaaS | 定期报告、递延收入/合同负债 |
| 半导体 | 公司公告、产业链调研、行业协会 |

## 行情与股本

可用：东方财富、新浪财经、腾讯行情、交易所行情页、公司公告中的股本。记录抓取日期；盘中价注明「盘中」。本技能不做估值，行情仅作事实引用。

## 日期验证

- 发布日原则上不超过 3 个月；超出时说明为什么仍适用（过时的机构预测必须标注已过时，不得作为现行锚）。
- 所有材料对应同一报告期；预期数据必须早于财报发布日。
- 每条来源能写出「名称 + 日期」。

## 禁止事项

- A 股标的禁引 SEC、EDGAR、10-Q、10-K、8-K、Form 等美股口径；港股/中概双重上市需在 meta.note 声明口径适配。
- 禁用「据了解」「市场普遍认为」「卖方一致认为」等无名表述。
- 禁用训练记忆中的财务数据。
- 禁用新闻报道作为财务事实或预期锚。
