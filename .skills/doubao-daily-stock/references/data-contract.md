## 数据字段与来源契约

## 核心原则

当前版本不获取 1 分钟级行情，不获取 30 个交易日日线，不获取 90 个交易日行情。所有数据只服务于对话版分析、飞书文档版摘要指标表、资金表、技术指标、价格区间、评级和风险判断。

数据准确性高于完整度。任一字段若无法从可靠来源核验，宁可留空，也不得猜测、补造或沿用疑似错误数据。

硬规则：

- 每一个进入报告的数值字段都必须先进入内部数据核验记录，并记录原始外部来源或 `seed_finance_search（同花顺数据库）` 工具来源标识。
- 不是 agent 自己计算的数值，必须有可打开来源链接或 `seed_finance_search（同花顺数据库）` 工具来源标识；对话版和飞书文档版中必须用来源编号引用该来源。
- 如果是自行计算，必须记录公式、输入字段、输入来源链接和计算过程；报告中应标注为“自行计算”或在口径说明中写明。
- 行情摘要、资金流、板块数据、MA、MACD、RSI、PE TTM、量比、投行一致预期等数据都属于需要来源的数据，不得以“行情接口数据”名义免引用。
- 金融数据优先使用 `seed_finance_search` 返回的明确结果；但摘要指标 8 项和 A 股资金流向指标是强制专用字段，只能来自 `seed_finance_search（同花顺数据库）` 或基于其返回值自行计算，不得使用 `general_search` 或其他来源兜底。量比只在收盘口径下自行计算，输入为 `seed_finance_search（同花顺数据库）` 返回的当日成交量和过去 5 个交易日成交量；若为盘中口径，量比写 `暂无`，不得用盘中成交量估算。港股/美股资金净流入/净流出、港股通南向资金若 seed 未返回，可使用 `general_search` 查找官方或权威机构来源，但数据日期必须是报告当日或最近一个有效交易日。其他金融数据未返回或口径不清时，才可使用 `general_search` 查找官方或权威机构来源。

## 股票身份

输出字段：

- `company_name`：公司全称或证券简称。
- `ticker`：标准代码，如 `300308.SZ`、`0700.HK`、`NVDA`。
- `exchange`：交易所。
- `currency`：人民币/港币/美元等。
- `report_request_time`：用户发起请求时间，盘中精确到分钟；对话版和飞书文档版开头必须统一使用该时间。
- `report_date`：报告生成日期，从 `report_request_time` 派生。
- `latest_valid_trading_day`：根据交易所日历和请求时间向前推算的最近有效交易日；非交易日、节假日、停牌时必须继续向前推。
- `data_as_of`：行情数据截至时间，盘中精确到分钟，收盘后写收盘日期和时间。
- `business_snapshot`：一句话业务定位，不超过 80 个中文字符。

股票身份也必须记录来源，例如交易所页面、公司公告、行情平台证券资料页或可信金融数据库。

## 内部数据核验记录

进入分析前必须先完成内部数据核验记录，作为后续分析和飞书文档版的事实底稿。该记录不要求保存为本地 `.txt`，也不得作为文末“信息来源”。建议字段：

- 股票身份：简称、代码、交易所、币种、行业/板块、原始外部来源链接。
- 时间口径：`report_request_time`、`latest_valid_trading_day`、`data_as_of`、是否盘中/收盘后/非交易日。
- 行情摘要：当前价/收盘价、涨跌幅、日内区间、总市值、PE TTM、成交额、换手率、量比；每项记录数值、单位、时间、来源名称、链接/工具来源标识、来源权威性判断。除量比外，来源必须为 `seed_finance_search（同花顺数据库）`；量比在收盘口径下记录为自行计算项，并记录当日成交量、过去 5 个交易日成交量、公式和输入来源 `seed_finance_search（同花顺数据库）`；盘中口径记录为缺失项，展示为 `暂无`。
- 资金流：按市场记录必需字段。A 股记录主力资金、超大单资金、大单资金，陆港通标的追加北向资金，来源必须为 `seed_finance_search（同花顺数据库）`；港股记录资金净流入/净流出，港股通标的可追加南向资金净流入/净流出；美股记录资金净流入/净流出。港股/美股资金字段先查 seed，未返回时可用 `general_search` 检索官方或权威机构来源，但数据日期必须是报告当日或最近一个有效交易日。A 股中单和小单暂不展示，也不进入资金流表。
- 板块数据：最相关板块、当日涨跌、涨跌家数、龙头/权重地位；每项记录来源名称、链接和时间口径。
- 技术指标：MA、MACD、RSI、震荡区间、压力位、支撑位；记录来源名称、链接、参数口径和是否允许展示具体读数。
- 新闻公告研报：来源、发布日期、标题、链接、是否在最近 3 日或报告要求区间内、与公司/行业关系。
- 一致预期：来源、字段、是否披露、是否自行计算、计算公式和原始输入来源。
- 缺失与剔除：所有冲突、可疑、无法打开、无法核验的数据和原因。

内部记录中未通过核验的数据不得进入报告正文、表格、综合判断理由或飞书文档版。

## 摘要指标表

飞书文档版顶部的 8 个指标：

1. 当前价格/收盘价：`1180.00元`
2. 涨跌幅：`+2.17%`
3. 日内区间：`1126.90-1196.16元`
4. 总市值：`约1.32万亿人民币`
5. 市盈率 TTM：`88x`
6. 成交额：`385.2亿元`
7. 换手率：`2.91%`
8. 量比：`1.2`

要求：

- 这 8 个摘要指标是强制专用字段，只能来自 `seed_finance_search（同花顺数据库）` 或基于其返回值自行计算，不得用 `general_search`、网页行情页、券商页面或其他数据库替代。量比只在收盘口径下为自行计算项，公式为 `当日成交量 / 过去 5 个交易日平均成交量`，输入数据必须来自 `seed_finance_search（同花顺数据库）`；盘中口径无法取得可比口径成交量，统一写 `暂无`。
- 飞书文档版使用 4 行 4 列 `lark-doc` 表格，表头为“指标 / 数值 / 指标 / 数值”，并用 `<colgroup>` 控制列宽；对话版可使用同字段的 Markdown 表格。
- 单位直接写进数值。
- 摘要指标表下方必须有一行注释，说明这些摘要指标的数据时间口径，并说明量比口径：收盘口径示例为 `注：以上行情摘要为 2026-06-12 收盘口径；量比为自行计算，公式为当日成交量 / 过去 5 个交易日平均成交量，成交量输入来自 seed_finance_search（同花顺数据库）。` 盘中口径示例为 `注：以上行情摘要截至 2026-06-12 13:30（交易日盘中）；量比因盘中缺少可比口径成交量，暂不计算。`
- 每个指标数值都必须带来源编号，例如 `1180.00元[1]`。
- 若多个指标来自同一个可打开行情页，可以共用同一来源编号；若来自不同数据页，必须分别编号。

飞书文档不依赖颜色表达涨跌或资金方向。涨跌幅、净流入/净流出必须在数值或正文中写清正负和方向。

## 资金交易线索与板块

A 股资金流字段：

- 所有 A 股资金字段只能来自 `seed_finance_search（同花顺数据库）`。
- 必须记录主力资金、超大单资金、大单资金的流入/流出情况；优先使用净流入/净流出口径，若数据源同时返回流入额和流出额，可写为 `净流入 +X（流入 X / 流出 X）`。
- 中单和小单数据暂不展示，不进入资金流表，也不作为复盘判断依据。
- 若股票属于陆港通标的，必须追加北向资金流入/流出或净流入/净流出口径；非陆港通标的不列北向资金。
- 每项必须有金额/比例、时间口径、来源标识和复盘含义，不得凭空编造。
- 资金流表格中的每个数值必须带来源编号；如果同一行多个数据来自同一来源，可在行内关键数值后引用一次。

港股资金流字段：

- 必须记录资金净流入/净流出；若数据源直接披露净买入/净卖出值，优先采用披露值并记录时间口径。
- 若股票属于港股通标的，且有权威来源披露，可追加南向资金净流入/净流出口径；非港股通标的不列南向资金。
- 港股资金字段先查 `seed_finance_search`；若未返回，可用 `general_search` 检索官方或权威机构来源，但数据日期必须是报告当日或最近一个有效交易日。仍无权威且时效合格的来源时写 `暂无`。
- 港股不分大单小单，不生成 `主力资金 / 超大单 / 大单 / 北向资金` 表格。

美股资金流字段：

- 必须记录资金净流入/净流出。
- 美股资金字段先查 `seed_finance_search`；若未返回，可用 `general_search` 检索官方或权威机构来源，但数据日期必须是报告当日或最近一个有效交易日。仍无权威且时效合格的来源时写 `暂无`。
- 美股资金流表格只保留一个资金类别：资金净流入/流出。
- 成交量相对均量、盘前/盘后表现、期权成交、ETF 流向、机构持仓变化可以作为补充交易线索写入“价格与量能”，但不得替代资金净流入/流出指标。

板块字段：

- 最相关的 1-3 个行业/概念/主题板块。
- 当日涨跌幅、涨跌家数或代表性龙头表现。
- 个股相对板块强弱：强于板块、弱于板块、跟随 beta 或独立 alpha。
- 板块名称、涨跌幅、涨跌家数和同业表现均需来源链接和来源编号。

## 技术指标

技术指标优先直接获取权威数据源披露值，不再要求本地重算。

字段：

- MA：MA5、MA10、MA20、MA60。
- MACD：DIF、DEA、MACD 柱体或数据源提供的等价口径。
- RSI：默认 RSI14；若来源使用其他参数，必须写明。
- 震荡区间、压力位、支撑位：来自行情平台技术位、券商/研报、可核验技术分析页，或有公式说明的自行计算。

要求：

- MA、MACD、RSI 三行固定。
- 每行包括：指标、标签、分析、判断。
- `标签` 和 `判断` 为 2-6 个字短语；`分析` 为 1-2 句完整解释，照顾不懂技术指标的读者。
- 技术指标读数必须有权威来源。若由 `seed_finance_search` 明确返回，且参数、时间口径和单位清楚，可以直接采用；若未返回，用 `general_search` 查找官方或权威技术指标/行情数据源。无法找到可信来源或口径不清时，不展示具体读数；技术指标表格单元格写 `暂无`，正文可简短说明“该指标暂未核验”，并降低技术面结论确定性。

价格区间：

- 震荡区间：一个区间，如 `402-446元`。
- 压力位：可以是单点、多个单点并列，或一个区间；不得写两个区间并列。
- 支撑位：可以是单点、多个单点并列，或一个区间；不得写两个区间并列。
- 飞书文档版不绘制价格区间带图；当前价、震荡区间、压力位、支撑位写入“关键价格区间”表格。
- 当前价、震荡区间、压力位、支撑位均属于数据字段，必须有来源编号；若由 agent 自行计算，必须引用输入数据来源并标明计算口径。

## 新闻、公告与信息来源

正文重要事实和所有展示数据必须能回到真实来源或明确工具来源标识：

- 最近 3 日内的公司公告、交易所/监管披露、公司 IR。
- 最近 3 日内的权威财经媒体、产业媒体、行业趋势新闻。
- 正文实际引用过的研报、一致预期或数据库页面。
- 当前价、涨跌幅、成交额、换手率、量比、资金流、板块、技术指标、PE TTM 等数据页面或 `seed_finance_search（同花顺数据库）` 结构化数据结果。

每条来源必须包含：

- 编号：`[1]`
- 媒体/来源名称
- 发表日期或数据日期
- 文章/公告/材料/数据页标题
- 行末右侧 `打开原文` 超链接；若为 `seed_finance_search（同花顺数据库）` 结构化数据，写明工具来源标识、数据日期/时间和字段口径，若工具返回可打开链接则同时附上链接。

链接要求：

- `打开原文` 必须是真实可打开的外部原文、公告、监管披露、媒体报道、行情页、数据库页或研报页链接；`seed_finance_search（同花顺数据库）` 强制专用字段可以使用工具来源标识作为来源，不得改用其他网页来源。
- 禁止空链接、`#`、`javascript:void(0)`、搜索结果页、虚构 URL。
- 正文每个来源编号 `[N]` 必须对应文末“信息来源”中的 `[N]` 来源行。
- 不得把多个不同事实都挂到同一个来源编号，除非这些事实确实来自同一篇原文或同一个数据页。
- 凡数据查不到来源，必须回到取数步骤；仍查不到则从报告删除。若固定表格必须保留该单元格，统一写短占位 `暂无`。

## 投行一致预期

优先数据源：

- `seed_finance_search` 优先查询一致预期、机构评级、目标价、盈利预测和估值数据。
- `general_search` 用于 `seed_finance_search` 未返回或字段不足时，检索可打开的券商研报摘要、评级页面、机构预测页面、公司/交易所披露或权威数据库页面。

字段：

优先根据市场和数据可得性选择一种结构：

1. 财务预测表：年份、归母净利润、营业收入、EPS、PE。
2. 评级目标价表：机构名称、最新评级、目标价、日期。
3. 无覆盖说明：无表格，仅说明暂无可核验投行/券商覆盖，并分析机会与风险。

适用规则：

- A 股或数据源披露完整盈利预测时，优先财务预测表，通常覆盖未来 3 年。
- 港股、美股或数据源主要披露券商评级/目标价时，优先评级目标价表，展示最近约 5 家机构。
- 不得把两套表头混在一张表里。
- 若经过 `seed_finance_search` 和 `general_search` 检索后，仍没有可核验的投行/券商盈利预测、评级或目标价，不得生成填满 `暂无` 的表格，也不得把公司 IR、公告或资讯聚合页当成机构覆盖。该节改为纯文字说明“暂无可核验覆盖”，并说明对买入该股票的机会和风险含义。

缺失处理：

- PE = 当前价格 / EPS。
- EPS = 归母净利润 / 总股本。
- 营收缺失不可臆造，表格内写 `暂无`。
- 评级或目标价缺失时表格内写 `暂无`。
- 目标价必须保留原始币种或明确换算币种。
- 同一机构多条记录只保留最新一条。
- 自行计算项必须在内部数据核验记录中记录公式，并在表格下方写简短口径说明。
- 一致预期表格中每一行关键数据必须有来源编号；若同一年预测或同一机构评级来自同一来源页，可共用同一来源编号。
- 无覆盖说明不需要强行角标；若引用外部页面证明无覆盖，则该页面必须可打开并进入“信息来源”。
