# 市场级问题:识别、转接、切片重点

先判断用户是否关注**市场整体影响**。本 Skill 只处理全市场、宏观/政策/新闻/风险事件、跨资产联动、资金风险偏好和市场情绪;不是具体公司、行业或板块的深度分析入口。无法判断时先澄清。

## 命中类型

`question_type` 只负责入口识别、时间窗和边界处理；具体怎么分析由 `references/analysis-playbook.md` 里的 Lens 组合决定。默认 Lens 是起点，不是硬规则。

| # | 类型 | 识别信号 | 默认时间窗 | 切片重点 |
|---|---|---|---|---|
| 1 | 全市场热点 / 交易主线 | "市场热点""交易主线""最近市场主线""A股主要热点" | 今天 / 近 24h;周度问题取近 5 个交易日 | 政策·行业扩散·指数/风格·量能·资金·情绪；按 `chat_contract.md` 写完整十模块正文 |
| 2 | 市场涨跌归因 | "今天市场为什么涨/跌""A股为什么下跌""港股为什么大涨" | 当日交易时段;盘前看隔夜外盘和今日预期 | 指数表现·风格切换·成交量·涨跌家数·领涨领跌方向·重要消息 |
| 3 | 宏观 / 政策事件影响 | "加息""降息""利率决议""央行政策""美联储/Fed""欧洲央行/ECB""PBOC/中国央行""通胀/就业数据""房地产支持政策""财政政策""监管政策""对权益市场影响""会不会跟随" | 政策发布起;必要时补近 7 天预期变化 | 政策原文·权威解读·影响变量·受影响环节·市场预期差 |
| 4 | 新闻 / 风险事件影响 | "地缘风险""突发新闻""风险升温""这条新闻意味着什么" | 事件发生起;突发默认近 24h | 事件事实·直接影响资产·避险链条·反向证据·后续验证 |
| 5 | 跨资产联动 | "美股大涨对港股影响""原油大涨对全球股市和通胀预期影响""美元/美债/黄金/能源联动""海外央行政策对本地市场影响""ECB加息对Fed和全球资产影响" | 事件发生起或近 3-7 天 | 起点资产·传导变量·受影响市场·历史可比·反向信号 |
| 6 | 风险偏好 / 市场情绪 | "风险偏好是不是变弱""资金情绪变化""市场情绪怎么样" | 今天 / 近 5 个交易日 | 成交·波动·涨跌停·行业扩散·资金流·避险资产·情绪指标 |
| 7 | 市场级更新追踪 | "这周变化""和上次比""最近新增了什么"且对象是市场整体/宏观事件 | 近 7 天 vs baseline | 只讲新增变化,不重讲背景 |

默认 Lens 组合见 `analysis-playbook.md`「问题类型 → Lens」表；上表 `#` 与 Lens 表一一对应。

## Few-shot:应触发

1. 最近一周 A 股市场的主要热点和交易主线有哪些?
2. 今天 A 股为什么下跌?
3. 美股大涨对港股有什么影响?
4. 央行降息对权益市场有什么影响?
5. 看到新闻说欧洲央行 6 月加息了,那是不是美联储后面也会跟着加息?
6. 最近市场主线是什么?
7. 布伦特原油价格大涨对全球股市和通胀预期有什么影响?
8. 国内房地产支持政策加码对 A 股整体市场情绪有什么影响?
9. 地缘风险升温对能源和黄金有什么影响?
10. 最近市场风险偏好是不是变弱了?

## 不触发并转接

| 用户意图 | 识别信号 | 处理 |
|---|---|---|
| 单只股票今天为什么涨跌 | 点名单只股票 + "今天为什么涨/跌/异动" | 转「个股日报」 |
| 某个板块今天为什么涨跌 | 点名板块 + "今天为什么涨/跌/领涨/领跌" | 转「板块热度分析」 |
| 某个行业长期怎么看 | 点名行业 + "长期怎么看/空间/格局/周期/投资价值" | 转「行业分析」 |
| 某家公司基本面如何 | 点名公司 + "基本面/商业模式/竞争力/估值/护城河" | 转「公司分析」 |
| 财报 / 业绩数据 | "财报""业绩预告""收入/利润""EPS""指引""电话会"且用户要数字或公司业绩解读 | 转「财报与业绩分析」 |
| 荐股 / 交易建议 | "推荐股票""能不能买""目标价""仓位""买卖点" | 拒绝投资建议;可改为市场级信息梳理 |

## 模糊场景澄清

当用户点名公司、行业或板块,但问题表述可能是在问市场整体传导时,先澄清:

> 你想看的是这件事对市场整体/相关资产的影响,还是想看这个公司/行业/板块本身的深度分析?

示例:
- "英伟达大涨怎么看?" → 可能是美股对科技股/港股/A股映射,也可能是公司分析;先澄清或按上下文判断。
- "房地产政策怎么看?" → 若问 A 股整体情绪,触发;若问房地产行业长期基本面,转行业分析。
- "CPO 最近怎么看?" → 若问市场主线/资金是否继续交易,可触发市场级主题;若问行业长期空间或板块热度,转行业分析/板块热度分析。

## 边界提示话术(命中越界时)

- 个股异动 → 「这个问题更适合用个股日报处理;本 Skill 只做市场整体和宏观事件影响解读。」
- 板块涨跌 → 「这个问题更适合用板块热度分析;本 Skill 只处理市场整体、政策/宏观/风险事件和资产联动。」
- 行业长期判断 → 「这个问题更适合用行业分析;本 Skill 不做行业长期基本面或投资价值判断。」
- 公司基本面 → 「这个问题更适合用公司分析;本 Skill 不做公司基本面、估值或目标价。」
- 财报业绩 → 「这个问题更适合用财报与业绩分析;本 Skill 不做公司财报数字解读。」
- 荐股/买卖点 → 「本 Skill 不提供买卖建议、目标价或仓位,只能做公开市场信息梳理。」

## 时间窗与交易时段

判定当前为盘前/盘中/盘后(A 股:09:30 前为盘前,09:30-15:00 盘中,15:00 后盘后;非交易日按最近交易日)。「今天大事」在盘前应偏「隔夜外盘 + 今日预期」,盘后偏「今日已发生 + 收盘解读」。时段口径写入开篇 `分析对象：` 行；飞书 XML 顶部元数据行从中解析 scope / time_window / as_of。
